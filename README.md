# Тайный факультет Волки

Платформа для сообщества факультета: достижения, звания, рейтинг, личные кабинеты.  
Стиль: минимализм, тонкий шрифт, тёмная тема.

## Требования

- PHP 7.4+ (с PDO MySQL)
- MySQL 5.7+ или MariaDB
- Веб-сервер (Apache/Nginx) или встроенный PHP: `php -S localhost:8000 -t C:\Users\Vlad\secret-faculty-wolves`

## Установка

### 1. База данных

Создайте базу в PHPMyAdmin (или через консоль):

```sql
CREATE DATABASE secret_faculty_wolves CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 2. Настройка подключения

Откройте `config.php` и при необходимости измените:

- **MySQL:** `DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASS`
- Если сайт открывается по адресу вида `http://localhost/secret-faculty-wolves/`, раскомментируйте в `config.php` строку `define('BASE_PATH', '/secret-faculty-wolves');`

### 3. Установка таблиц и админа

В браузере откройте:

```
http://ваш-сайт/secret-faculty-wolves/install.php
```

Скрипт создаст таблицы и учётку администратора:

- **Логин:** `admingross`
- **Пароль:** `admingross`

После успешной установки **удалите или переименуйте** `install.php`.

### 4. Запуск (локально)

Из папки проекта:

```bash
php -S localhost:8000
```

Откройте в браузере: `http://localhost:8000`

## Структура

| Раздел | Описание |
|--------|----------|
| `/` | Главная: градиент, «Тайный факультет Волки», кнопки Войти / Зарегистрироваться |
| `/auth/login.php` | Вход |
| `/auth/register.php` | Регистрация (имя, фамилия, дата рождения, логин, пароль) |
| `/cabinet/` | Личный кабинет: данные, звание, «обо мне», достижения, кнопка «Изменить» |
| `/rating/` | Рейтинг по званиям, переход в профиль участника |
| `/profile/view.php?id=N` | Публичный профиль участника |
| `/admin/` | Админка: список участников, назначение званий и достижений, создание достижений (эмодзи + название) |
| `/admin/ranks.php` | Редактор званий: тип (полоска/звезда/бейдж), цвет, количество, порядок |

## Подключение к PHPMyAdmin

1. Запустите MySQL (XAMPP, OpenServer, Denwer и т.п.).
2. Откройте PHPMyAdmin, создайте БД `secret_faculty_wolves` (если ещё не создана).
3. При желании можно импортировать только схему: `sql/schema.sql` (админа тогда создаёт `install.php`).

## Безопасность

- Пароли хранятся в виде хеша (bcrypt).
- Для продакшена смените пароль админа после первого входа и настройте HTTPS.
