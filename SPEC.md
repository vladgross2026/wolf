# Спецификация: Платформа «Тайный факультет Волки»

## Цель
Веб-платформа для сообщества (факультета) с учётом достижений, званий и «легенд» участников. Стиль: минимализм, современный, тонкий шрифт. Бэкенд: PHP + MySQL (PHPMyAdmin).

---

## 1. Начальный экран (Landing)
- **Фон:** чёрный с красивым градиентом (например, тёмно-серый → чёрный → глубокий синий/фиолетовый).
- **Анимация:** плавное движение/пульсация градиента (CSS animation или лёгкий JS).
- **Текст:** «Тайный факультет Волки» — крупно, тонкий шрифт, контрастный цвет (белый/светло-серый).
- **Кнопки:** «Войти», «Зарегистрироваться» (и при необходимости «О платформе» и т.д.) — минималистичные, в едином стиле.

---

## 2. Регистрация и вход
- **Регистрация:** имя, фамилия, дата рождения, логин (или email), пароль (с подтверждением). Валидация на клиенте и сервере.
- **Вход:** логин + пароль. Сессии PHP, защита от простых атак (хеш паролей, подготовленные запросы).
- **Хранение:** таблицы `users` (id, login, password_hash, first_name, last_name, birth_date, created_at, is_admin), при необходимости отдельная таблица сессий.

---

## 3. Администратор
- **Учётная запись:** логин `admingross`, пароль `admingross` (при первом запуске создать через миграцию/сид или проверку в коде; позже пароль лучше сменить).
- **Права:** только пользователь с `is_admin = 1` имеет доступ к админ-разделу.
- **Функции админа:**
  - Назначение участникам званий и достижений.
  - Создание и настройка званий (редактор звания — см. п.6).
  - Создание достижений (эмодзи + название), привязка к пользователям.

---

## 4. Личный кабинет
- **Данные:** имя, фамилия, дата рождения, логин; кнопка «Изменить» (редактирование без смены логина или с ограничениями).
- **Звание:** отображается название и иконка звания (из редактора званий).
- **«Обо мне» / «Моя история»:** одно текстовое поле (или несколько), которое пользователь заполняет сам; отображается в профиле.
- **Достижения:** список достижений, выданных админом (эмодзи + название); отображаются списком/чипами, суммируются (все свои достижения в одном блоке, новые дополняются рядом).

---

## 5. Вкладка «Рейтинг»
- Список участников по ранжиру (по званию: порядок званий задаётся при создании звания; при равенстве — по количеству достижений или по дате регистрации).
- Для каждого: аватар/иконка (опционально), имя, звание, краткие данные. Клик по участнику — переход на его профиль (публичный вид).
- **Публичный профиль:** вкратце данные (имя, звание, «обо мне»/история), достижения (эмодзи + название). Без кнопки «Изменить» и без доступа к смене пароля.

---

## 6. Редактор званий (только для администратора)
- Админ создаёт звание: название, тип отображения (полоска, звезда, другая форма), цвет, количество (например, количество звёзд или уровень полоски), порядок для рейтинга.
- Внешний вид — в стиле рангов из Counter-Strike: аккуратная иконка/бейдж (CSS + при необходимости простые SVG/иконки). Редактор сохраняет параметры в БД (таблица `ranks`: id, name, type [bar|star|badge], color, amount/level, sort_order, icon_css или аналогично).
- Пользователям назначается одно текущее звание (поле в `users`: rank_id).

---

## 7. База данных (MySQL, PHPMyAdmin)
- **Таблицы:**
  - `users`: id, login, password_hash, first_name, last_name, birth_date, about_me (текст истории), rank_id, is_admin, created_at, updated_at.
  - `ranks`: id, name, type, color, amount, sort_order, style_json (или отдельные поля для вида).
  - `achievements`: id, emoji, name, created_at (глобальный справочник достижений).
  - `user_achievements`: user_id, achievement_id, assigned_at (связь пользователь–достижение).
- Подключение через PDO, конфиг в отдельном файле (config.php или .env), удобно править под PHPMyAdmin/XAMPP/OpenServer.

---

## 8. Общий стиль
- Минимализм: мало элементов, много «воздуха», чёткая сетка.
- Современный: скругления, тени по необходимости, градиенты только где указано.
- Шрифт: тонкий (font-weight: 200–300), без засечек (например, Inter, Roboto, Montserrat или аналог).
- Цвета: тёмная тема на основе чёрного/тёмно-серого, акценты — один цвет (например, приглушённый золотой/белый для кнопок и званий).

---
